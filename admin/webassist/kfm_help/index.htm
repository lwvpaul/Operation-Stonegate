File Manager Help

Print this page

The File Manager allows you to upload, select, and modify files. Image files that have been uploaded using the File Manager can be cropped, rotated, and resized.

If you are simply selecting a file, you can double-click the file, or select the desired file and click OK.

Uploading a File

Follow these steps to upload a file.

  1. If available, choose the directory to upload to from the Directories pane.
  2. Select Choose File.
  3. Browse for the file you wish to upload.
  4. Click OK.
  5. In the File Manager, click Upload.

Once uploaded, the file will appear within the File Manager.

Cropping an Image

Follow these steps to crop an image.

  1. From within the File Manager, select the Image you wish to crop.
  2. Choose the crop option from the Actions toolbar.
  3. When the image opens in the crop dialog, click and drag over the area you wish to keep.
  4. With the desired area selected, choose how you wish to save this file:
    1. Crop: Will overwrite the original file with the cropped version.
    2. Crop to new file: Will create a new file preserving the original.

Your cropped image will appear within the File Manager.

Rotating an Image

Follow these steps to rotate an image.

  1. From within the File Manager, select the Image you wish to rotate.
  2. From the Actions toolbar, choose the desired rotate option.
    1. Rotate clockwise: will rotate the image 90 degrees in a clockwise direction.
    2. Rotate counter-clockwise: will rotate the image 90 degrees in a counter-clockwise direction.

The image will be rotated instantly.

Resizing an Image

Follow these steps to resize an image.

  1. Select the image you wish to resize.
  2. Choose the resize option from the Actions toolbar.
  3. Specify the desired width in the dialog that appears and click OK.
  4. Specify the desired height in the dialog that appears and click OK.
  5. Confirm that the new dimensions are correct and click OK.

Your resized image will be available from the File Manager window.

Note: The rotate image options may not be available on some installations.
